Kicksharing — Анализ данных и прогнозирование спроса

Описание: Проект посвящён анализу и прогнозированию спроса на сервисы кикшеринга. Включает исследование данных (EDA), обработку признаков и построение моделей машинного обучения для предсказания объёма аренды самокатов. Используются Python, pandas, scikit-learn, визуализация и бустинг-модели.

1. Источник данных

Данные взяты из открытого источника HSE DANO: https://dano.hse.ru/mirror/pubs/share/987942251.pdf

2. Цели проекта

Провести исследовательский анализ данных (EDA) для понимания факторов, влияющих на спрос;

Выполнить очистку и предобработку данных, создать производные признаки;

Построить и сравнить модели прогнозирования (линейные, бустинг, деревья решений);

Оценить качество моделей с помощью метрик MSE, RMSE;

Сформировать воспроизводимую структуру проекта.
